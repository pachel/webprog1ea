Web Storage

A Web Storage egy HTML5 API, amely lehetővé teszi, hogy az adatok a böngészőben tárolódjanak. Két típusa van: localStorage és sessionStorage.

  • localStorage: Az adatok tartósan tárolódnak, még a böngésző bezárása után is elérhetők.
  • sessionStorage: Az adatok csak az aktuális böngésző munkamenet során érhetők el, és a böngésző bezárásakor törlődnek.

Az alábbi példában a localStorage használatával mentheted el az adatokat, és később betöltheted őket.

Web Workers

A Web Workers egy HTML5 API, amely lehetővé teszi, hogy JavaScript kódot külön szálon futtassunk. Ez azt jelenti, hogy a hosszabb ideig tartó műveletek (például számítások) nem blokkolják a fő szálat, így a felhasználói felület továbbra is reszponzív marad.

A Web Workers használatához létre kell hozni egy új worker objektumot, amely egy külön JavaScript fájlban található kódot futtat.

Az alábbi példában egy egyszerű Web Worker-t indítunk, amely üzenetet küld vissza a fő szálnak.
